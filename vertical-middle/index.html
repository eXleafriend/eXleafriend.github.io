Vertical Middle

When you know height of block

It works on IE 7.

CSS:

.vertical-middle {
  height: 10em;
  background-color: #ffe0e0;
  position: relative;
}
.vertical-middle > .vertical-middle-item {
  background-color: #e0ffe0;
  position: absolute;
  top: 50%;
}

HTML:

You shold specify height of block & margin-top as half of height with style.

<div class="vertical-middle">
  <div class="vertical-middle-item" style="

    height: 1.5em;
    margin-top: -0.75em;
  ">
    Hello, Middle!
  </div>
</div>
